
# Game of Life

A simple demo of Conway's Game of Life written in react and typescript.

## Run Locally

Clone the project

```bash
  git clone git@github.com:jacobgad/game-of-life.git
```

Go to the project directory

```bash
  cd game-of-life
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
